# Nishant Bhadke Command Portfolio

This is the separate V2 portfolio for Nishant Bhadke. It keeps the first portfolio untouched and explores a more interactive command-driven experience.

Live site:
[nishantbhadke.github.io/nishant-bhadke-command-portfolio](https://nishantbhadke.github.io/nishant-bhadke-command-portfolio/)

V1 remains separate:
[nishantbhadke/nishant-bhadke-portfolio](https://github.com/nishantbhadke/nishant-bhadke-portfolio)

## Interaction Model

The portfolio can be navigated with typed commands, quick command chips, project carousel controls, and an on-screen typewriter keyboard.

Useful commands:

```text
help
about
work
projects
skills
resume
contact
rbl-bcms
rbl-radc
suryoday
cub
clear
```

## Run Locally

```powershell
npm install
npm run dev
```

Static preview:

```powershell
npm run build
npm run preview
```

Open `http://127.0.0.1:4174`.

## Project Split

- V1: `nishant-bhadke-portfolio`
- V2: `nishant-bhadke-command-portfolio`

Use `V1` or `V2` in future requests so changes go to the intended portfolio.
